Getting Started with JavaScript

Strings - A string is a sequence of characters

a-z, space, number characters etc.

Example: "this is a string"

Strings have a length property, the number of characters in the string. For example to find the length of a string:
"this is my string".length

Numbers - integers (..., -2, -1, 0, 1, 2, 3, ...) or decimal

Calculations can be made using numbers: 3+4;

Use * for multiplication, / for division

Boolean - true / false (yes/no):

12 > 100; // false (no)
"this" == "that" "This sentance is longer than 10 characters!" > 10;

Operators

> greater than
< less than
<= less than or equal to
>= greater than or equal to
=== equal to
!== not equal to

Comments

Start a line with //, for example:
// this line is a comment

Getting a user response

confirm() can be used to get a true / false response:
confirm('My message');

prompt() can be used to get a string:
prompt("What is your name?");
