Session 2

What we did

We started the Kahn Academy Computer Programming course - Introduction to JS (JavaScript): Drawing and Animation. First we watched the "What is programming video". Next we watched video and worked through the challenges in "Drawing Basics".

First we made a simple snowman, for example:
ellipse(200,350,100,100);
ellipse(200,300,60,60);
ellipse(200,265,40,40);
Note that the ellipses are drawn in sequence, with the first one drawn behind the next etc.

Then we experimented with rectangles and lines.

We learnt that all statements (commands) are on a line and end in a semi-colon ;

We learnt that output can be sent to the console log. The console log is a way of getting information from a program as it is running.

For example: console.log("This is a test message");

We looked at how colors on web pages can be set using the RGB (Red-Green-Blue) color model.

Processing JS Shape Functions (Commands)

ellipse(x,y,width,height);

x: distance of center going right from the top left corner
y: distance of center going down from top left corner
width: overall width of ellipse
height: overall height of ellipse

rect(x,y,width,height);

x and y are to the top corner!
x: distance of top left rectangle corner going right from the top left corner
y: distance of top left rectangle corner going down from top left corner
width: overall width of rectangle
height: overall height of rectangle

fill(r,g,b);

fill(); sets the color for following shape commands
r,g,b color numbers are between 0 and 255
r Red
g Green
b Blue
