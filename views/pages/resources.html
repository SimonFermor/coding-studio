Resources

During the studio sessions students will be using a number of websites and it will be useful if students have accounts on the following sites. It would be helpful to create accounts before the first session.

These sites generally do not provide social interaction and so are not age restricted. All these sites have a free membership level.

Codecademy The Codecademy provides online lessons and practice in web technologies.
GitHub GitHub is a site for storing code and related files.
jsFiddle jsFiddle is an editor for creating and interacting with JavaScript.
Khan Academy The Kahn Academy has an extensive set of online courses. There is a useful course on JavaScript.
Mozilla Thimble Mozilla Thimble provides an interactive environment for working with web based documents
Lucid Chart Lucid chart provides online charting, such as for creating flow charts
