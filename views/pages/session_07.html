Session 7

This week we used the Blockly Games website to learn programming concepts.

Blockly is used in a number of systems for block based coding. The system allows programs to be constructed by draging and dropping program elements equivalent to program statements. The Blockly Games website shows generated JavaScript and so reinforces much of what we have already covered.

The same block based system is used in other tools such as the Google App Inventor
