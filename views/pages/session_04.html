Session 4

The theme for the fourth session was variables and binary numbers.

We started by talking about variables and how they can be used. Variables are named locations where values can be stored and can then be referenced by name in a program.

We played a game in which each student played the role of a variable and we explored using the variables in an algorithm for calculating and printing values in the Fibonacci series.

The algorithm has been converted to JavaScript

We watched the Kahn Academy video on Variables and worked through the challenges.

Finally, we talked about binary numbers and worked through some examples of binary addition on the board. We discussed the binary growth sequence 1,2, 4, 8, 16, 32, 64 etc. and why binary is important when computers use storage registers that can only be on or off, or store 0 or 1.
